# Getting Started with Wixen Mail

This page is intentionally brief and points to canonical docs to avoid duplication.

## Prerequisites

- Rust 1.70+
- Git

## Build and Run

```bash
git clone https://github.com/PratikP1/Wixen-Mail.git
cd Wixen-Mail
cargo build
cargo run --bin ui_integrated
```

## Validate Locally

```bash
cargo test --quiet
cargo build --quiet
```
